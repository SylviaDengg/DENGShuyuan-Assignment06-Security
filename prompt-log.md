# Prompt Log

## Purpose

Use this file to store the full AI prompt history required by the assignment.

## Suggested Structure

### 1. Original Assignment Prompt

Paste the assignment instructions here.

### 2. Prompts Sent To The AI Tool

Paste each prompt used during development.

### 3. AI Responses

Paste the corresponding AI outputs.

### 4. Notes On Changes Made

Summarize what was implemented or revised after each prompt.
